# Readme

This example was created during a very random test of ChatGPTs performance of transcribing an image of sheet music into MusicXML.

The input was the first page of Franz Schubert's "Der Tod und das Mädchen" from its first edition:
Vienna: Diabelli, n.d.[1822]. Plate D. et C. No. 855.
Available online at [Eda Kuhn Loeb Music Library, Harvard University](https://curiosity.lib.harvard.edu/digital-scores-and-libretti/catalog/37-990016346520203941)

The MusicXML in ChatGPTs answer was pasted into to MusicXML file provided in this folder. The MEI file was converted with Verovio using [mei-friend](https://mei-friend.mdw.ac.at/).

The conversation with ChatGPT was obtained via ChatGPT's Data export (available via Settings -> Data Controls -> Export data).
